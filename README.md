# 🚀 Guía de instalación y primeros pasos con Playwright

Playwright es un framework de pruebas end-to-end para aplicaciones web modernas.  
Permite automatizar navegadores (Chromium, Firefox y WebKit) en modo **headless** o con interfaz gráfica.

---

## 📦 Requisitos previos
- **Node.js**: versión 20.x, 22.x o 24.x 
- **Sistemas operativos soportados**:
  - Windows 11+, Windows Server 2019+ o WSL 
  - macOS 14 (Sonoma) o superior 
  - Linux: Debian 12/13, Ubuntu 22.04/24.04 (x86-64 o arm64) 

---

## ⚙️ Instalación

### 1. Crear un nuevo proyecto con Playwright
```bash
npm init playwright@latest
```
